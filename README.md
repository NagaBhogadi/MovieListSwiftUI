# SwiftUI Movie App

A simple Movie App built using SwiftUI that displays a list of movies and detailed information about each movie.

## Features

- Display movie list using SwiftUI List
- Navigate to movie details using NavigationStack
- Show movie poster images
- Display movie title, genre, rating, and release year
- Show detailed movie description
- Reusable SwiftUI components
- Clean and modern UI
